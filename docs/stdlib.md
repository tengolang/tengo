# Standard Library

- [os](https://github.com/tengolang/tengo/blob/main/docs/stdlib-os.md):
  platform-independent interface to operating system functionality.
- [text](https://github.com/tengolang/tengo/blob/main/docs/stdlib-text.md): regular
  expressions, string conversion, and manipulation
- [math](https://github.com/tengolang/tengo/blob/main/docs/stdlib-math.md):
  mathematical constants and functions
- [times](https://github.com/tengolang/tengo/blob/main/docs/stdlib-times.md):
  time-related functions
- [rand](https://github.com/tengolang/tengo/blob/main/docs/stdlib-rand.md):
  random functions
- [fmt](https://github.com/tengolang/tengo/blob/main/docs/stdlib-fmt.md):
  formatting functions
- [json](https://github.com/tengolang/tengo/blob/main/docs/stdlib-json.md): JSON
  functions
- [enum](https://github.com/tengolang/tengo/blob/main/docs/stdlib-enum.md):
  Enumeration functions
- [hex](https://github.com/tengolang/tengo/blob/main/docs/stdlib-hex.md): hex
  encoding and decoding functions
- [base64](https://github.com/tengolang/tengo/blob/main/docs/stdlib-base64.md):
  base64 encoding and decoding functions
- [coro](https://github.com/tengolang/tengo/blob/main/docs/stdlib-coro.md):
  coroutines: pauseable functions that preserve local state across yields
